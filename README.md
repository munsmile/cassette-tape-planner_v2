# Cassette Tape Planner

카세트 테이프 녹음 시 A면/B면의 사용 시간과 남은 시간을 계산하는 React 앱입니다.

## 주요 기능

- A면/B면별 곡 목록 관리
- 테이프 총 길이 기준 각 면의 남은 시간 계산
- MP3/FLAC 파일 드래그 앤 드롭
- 파일명 기반 제목 자동 입력
- 음원 메타데이터 기반 재생 시간 자동 입력
- 수동 곡 추가 및 시간 수정

## 실행 방법

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 배포

Netlify, Vercel, GitHub Pages 등에 배포할 수 있습니다.
